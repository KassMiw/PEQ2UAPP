# PEQ2UAPP
A bash script to convert `ParametricEQ.txt` to `UAPP Preset.xml` ToneBoosters 10-bands ParametricEQ. 

## How to use? 
Save all `ParametricEQ.txt` on the same directory with `PEQ2UAOP.sh`
Just run... 
```
bash PEQ2UAPP.sh
```
Wait until finished. 
All *xml file will be saved in `TBEQPresets` folder.
